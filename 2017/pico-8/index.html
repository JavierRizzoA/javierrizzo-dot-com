Pico-8

Hola, en esta página puedes encontrar Pico-8, programa que usamos para el taller de programación de videojuegos en El Garage, así como un poco de información adicional.

Se trata de una "consola de fantasía" para hacer (en lenguaje Lua), compartir y jugar pequeños juegos y otros programas de computadora.

Pico-8 es un software de paga, pero una sola licencia puede ser usada en varias computadoras debido a que no tiene DRM, por eso es que lo puedes descargar como parte del taller. Si lo utilizas con otros propósitos, considera comprar tu propia licencia (existen descuentos del 80% para estudiantes de talleres de Pico-8).
